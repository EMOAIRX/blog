三分钟了解希尔伯特空间

2022-06-27 0 Comments

一，线性空间

各种矩阵

厄密矩阵 正规矩阵 酉阵 置换矩阵 正定矩阵

线性与非线性

特征值啥的

矩阵谱半径

\(A^m\) 收敛于 \(0 \iff\) 谱半径小于 \(1\)

求导

矩阵可以求导，每个元素求导

也可以矩阵对矩阵求导，类似雅可比矩阵。

矩阵求导积分就是每个元素自己做。

矩阵函数

把常见的级数换成矩阵。

利用方阵的幂级数可以定义矩阵函数。

二、距离空间

是一个拓扑的概念

非负性，对称性，三角不等式

利用Holder不等式，Cauchy不等式

1、 \(l^p\) 空间（\(l^\infin\) ） \[ l^p = \set{x=(x_1,x_2\dots x_n\dots) \mid \sum_{n=1}^{\infin} \abs {x_n}^p < +\infin} \]

\[ \rho(x,y)=(\sum_{n=1}^{\infin} \abs{x_n-y_n}^p)^{\frac{1}{p}} \] 元素定义在自然数集合上，反映出来是一个序列，相当于

2、\(L^p\) 空间：定义在可测集上的空间 \[ L^p(E)=\set{f:f可测,\int_E \abs {f}^p \rm{dx}<+\infin} \] 对于 \(1\le p <\infin\)

对于 \(L^\infin\) ，则类似用 esssup 定义

Holder不等式的一个性质：\(f\in L^p(E),g\in L^q(E)\)，则 \(f\cdot g\in L^1(E)\)

def 收敛

在某个距离空间，按照某个

def 稠密与可分

稠密：如果 \((X,\rho)\) 是距离空间， \(A,B \subseteq X\) ，若对于 \(\forall x \in B\)，\(\exist \set{x_n} \subseteq A\) ，都有 \(x_n\) 收敛到 \(x\)，则是稠密的。

\(\Q\) 在 \(\R\) 中稠密

P[a,b] C[a,b]

(P[a,b],rho) 在 (C[a,b],rho) 中稠密

可分：如果距离空间存在一个可数的稠密子集

def 柯西列

$(x_n,x_m) $ as \(n,m \to \infin\)

\(\set{r_n=(1+\frac{1}{n})^n} \subseteq \Q\) 也是柯西列

def 完备距离空间

\((X,\rho)\) ，如果其中任一柯西列都是收敛列，那么 \((X,\rho)\) 就是完备距离空间

def 等距映射

如果两个空间 \((X,\rho)\) , \((X_1,\rho_1)\)

存在一个双射，保持距离不变

theorem 距离空间完备化

任何一个距离空间都可以嵌入到一个完备空间

theorem 压缩映射原理

Banach 空间

def 赋范线性空间

定义范数： 1、\(\norm{x}\ge 0\)

2、\(\norm {\alpha x}=\abs \alpha \norm{}\)

3、|x+y|<=|x|+|y|

\(V,\Vert\cdot \Vert\)

可以变成距离空间 \(\rho(x,y) = \norm {x-y},\forall x,y,\in V\)

一范数收敛就是一致收敛

\(L^p\) 定义范数

\(\norm {f}_p=\)

def Banach空间

如果 (V,||) 是赋范线性空间，作为距离是完备的，那么 (V,||) 是Banach空间

theorem 完备化定理

赋范线性空间可以完备化变成Banach空间

Hilbert空间

def 内积空间

1、关于第一个变量线性

2、对称性（复的话要共轭）

3、非负性，自己和自己大于0，

\((V,(.))\) 就是内积空间

可以定义 \(\norm {x}=\sqrt{(x,x)}\)

所以内积空间可以诱导赋范线性空间

内积和范数有关系，一定满足

Theorem 平行四边形法则

\[ \norm {x+y}^2+\norm {x-y}^2=2 \norm{x}^2 + 2\norm{y}^2 ,\forall x,y,\in V \]

如果范数满足这个不等式，就可以定义内积 \[ P=R:&(x,y)=\frac{1}{4}(\norm {x+y}^2-\norm {x-y}^2)\\ P=C:&(x,y)=\frac{1}{4}(\norm {x+y}^2-\norm {x-y}^2+i\norm {x+y}^2-i\norm {x-y}^2)\\ \]

def Hilbert空间

完备的内积空间称为 Hilbert空间

theorem 完备化定理

内积空间可以完备化，变成希尔伯特空间

def 直交

如果两个元素内积为 \(0\) ，那么这两个元素是直交的。

同时也可以是集合和集合垂直。

性质

(a).如果两个元素两两直交，那么就有勾股定理。

(b).如果和稠密子集垂直，那么只有0向量。

(c).如果 \(M\in V\)，\(M^\perp\) 自动是 \(V\) 的闭子空间

theorem 直交分解

任何一个 \(V\) 的闭子空间 \(M\) ，那么任何一个向量都可以唯一分解成 \(M\) 和 \(M^\perp\) 的

def 标准直交系

两两垂直，长度为1。\(e_n\) 是标准直交系
